## extern

如果是C++编译器，以下的代码按照C编译

```c
#ifdef __cplusplus
extern "C"{
#endif

void testCfun();

#ifdef __cplusplus
}
#endif
```

## 栈

```c
初始化: int stk[N] tt = 0
插入push: stk[++t] = x
弹出pop:  t--
判空empty:  tt > 0 不空
栈顶top: stk[tt]
```

## 队列

```c
初始化：int q[N], hh = 0, tt = -1 
判空: hh <= tt 不空
插入: q[++tt] = x
弹出: hh++
队头: q[hh]
```
